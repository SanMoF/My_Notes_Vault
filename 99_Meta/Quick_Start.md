# Quick Start

## Simple Daily Workflow

1. **Take notes in class** → Save to `10_Courses/Active/[Course Name]/`
2. **Have an idea** → Drop it in [[00_Inbox]]
3. **Start a build** → Use [[Project]] template in `20_Projects/Active/`
4. **Once a week** → Clean up Inbox, move finished stuff

That's it.

## When to Use Each Template

| Template | Use When | Example |
|----------|----------|---------|
| **Course_Note** | Starting a new class | MECH 301 - Control Systems |
| **Project** | Building something | Line Follower Robot |
| **Formula_Sheet** | Making a cheat sheet | PID formulas for exam |

## Folder Structure

```
My_Vault/
├── 00_Inbox/          ← Temporary notes
├── 10_Courses/        ← Class notes
│   ├── Active/
│   ├── Completed/
│   └── Reference/
├── 20_Projects/       ← Builds and hands-on stuff
│   ├── Active/
│   └── Completed/
├── 30_Knowledge/      ← Things you look up often
│   ├── Concepts/
│   ├── Formulas/
│   └── Programming/
├── 40_Resources/      ← PDFs, papers, reference material
│   ├── Books/
│   ├── Papers/
│   └── Websites/
└── 99_Meta/           ← Templates and guides
```

## Weekly Cleanup (5 minutes)

- [ ] Move notes from Inbox to proper folders
- [ ] Mark completed projects → move to Completed
- [ ] Archive old courses

## Naming

- **Courses:** `COURSE-XXX - Course Name`
- **Projects:** `Project - Name`
- **Notes:** Just describe what it is

---
Don't overthink it. Just start taking notes.
